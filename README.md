# To-do Pomodoro App

A retro-styled To-Do List with an integrated Pomodoro Timer built with **HTML, CSS, JavaScript**, and **Electron**.  
It allows users to manage tasks while keeping track of work and break intervals.

---

## Features

### To-Do List
- Add tasks with a simple input field.
- Remove tasks individually.
- Scrollable task list for easy management.
- Retro pixel font style for a classic look.

### Pomodoro Timer
- Select work duration (e.g., 25 or 50 minutes).
- Select break duration (5 or 15 minutes).
- Timer automatically switches to break after work ends.
- Custom popup notification for break completion.
- Start and reset buttons for flexible timing.
